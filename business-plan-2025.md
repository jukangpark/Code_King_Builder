# 📋 Code King Builder 사업계획서 (2025)

## 1. 회사 개요

### 1.1 사업명

**Code King Builder – AI 기반 웹사이트 빌더 SaaS 플랫폼**

### 1.2 비전

> “누구나 AI와 대화하듯이 전문적인 웹사이트를 만들 수 있는 세상”

### 1.3 미션

- 비개발자도 쉽게 웹사이트 제작 가능하도록 지원
- 한국 시장에 특화된 로컬 서비스(카카오·네이버·토스) 연동
- 스타트업·소상공인을 위한 빠른 MVP 제작 및 배포
- AI + 전문가 협업을 통한 고품질 웹사이트 제공

---

## 2. 시장 분석

### 2.1 글로벌 시장

- 웹사이트 빌더 시장: 약 **2.6조 원 (2024)** → **5.1조 원 (2033)**
- 로우코드/노코드 시장: 약 **36조 원 (2024)** → **237조 원 (2033)**

### 2.2 국내 시장

- 한국 로우코드 시장: 약 **7,100억 원 (2023)** → **3.3조 원 (2030)**
- SMB, 스타트업, 프리랜서 창업자 대상 **웹사이트/내부툴 수요 급증**

### 2.3 고객 세그먼트

1. 창업자·프리랜서 – 랜딩페이지, MVP 제작
2. 전통 SMB – 예약/결제 시스템, 내부 대시보드 구축
3. 스타트업 비개발 직군 – 빠른 프로토타이핑 및 마이크로앱 제작

### 2.4 경쟁 우위

- 🇰🇷 한국 특화 로컬 연동 (카카오 로그인, 네이버 지도, 토스 결제 등)
- 🤖 AI 프롬프트 엔지니어링 최적화 (한국어 UX)
- ⚡ 즉시 배포 시스템 (Vercel + Docker)
- 🧩 MCP 기반 템플릿 관리 (AI + 템플릿 연동 자동화)

---

## 3. 제품 및 서비스

### 3.1 핵심 기능

- AI 대화형 웹사이트 빌더 (자연어 입력 → 코드/사이트 자동 생성)
- 업종별 템플릿 라이브러리 (10종 → 50종)
- 실시간 미리보기 & 원클릭 배포 (Vercel, Netlify 지원)
- 한국형 서비스 연동 (카카오, 네이버, 토스, 카카오페이)
- 전문가 검토 서비스 (AI 생성물 + 개발자 리뷰)

### 3.2 기술 아키텍처

- **Frontend**: Next.js 15, React 19, TailwindCSS
- **AI Layer**: OpenAI GPT-4, Claude Sonnet + MCP Adapter
- **DB/인증**: Supabase (PostgreSQL, Auth)
- **배포**: Vercel / Docker / GitHub Actions
- **UI/UX**: Framer Motion, HeadlessUI

### 3.3 차별화 포인트

- 글로벌 빌더(Wix, Webflow)는 한국 로컬 연동 미비
- Code King Builder는 **AI + 한국형 UX 최적화**로 차별화

---

## 4. 비즈니스 모델

### 4.1 수익 구조

Code King Builder의 수익 구조는 **일회성 패키지 딜리버리**(완성형 사이트)와 **구독형 SaaS**(지속 사용/관리) 두 축으로 구성됩니다.

#### 💰 일회성 패키지

1. **STANDARD – ₩99,000**

   - AI 빌더 기본 웹사이트 제공
   - 컬러/브랜딩 반영, Vercel 자동 배포

2. **DELUXE – ₩399,000**

   - STANDARD 포함 + 개발자 커스텀 (최대 8시간)
   - 추가 기능: 폼, 블로그, 예약/문의 기능
   - 반응형 최적화 + SEO 기본 설정

3. **PREMIUM – ₩990,000**
   - DELUXE 포함 + 고급 커스텀 (최대 20시간)
   - 결제 연동(토스페이, 카카오페이), 회원가입/로그인, 대시보드
   - 보안 강화(JWT, Supabase Auth)
   - 유지보수 1개월 무료

#### 🔄 구독형 SaaS

- 월 **₩20,000 (2주 무료 체험)**
- 지속적인 웹사이트 사용 및 관리 제공

### 4.2 1년차 예상 수익

- 일회성 패키지: 약 **10억 원**
- SaaS 구독: 약 **14.4억 원**
- 유지보수: 약 **6억 원**
- **총 매출: 약 30억 원**

## 5. 마케팅 전략

### 5.1 채널별 접근

- 창업자: 인스타그램 숏폼 광고, 카카오톡 채널, 네이버 검색광고
- SMB: 업종별 커뮤니티, 상공회의소 협력, 전통시장 디지털화 프로그램
- 스타트업: 디캠프, 구글 스타트업 캠퍼스, 테크 컨퍼런스

### 5.2 콘텐츠 전략

- 블로그: **AI 웹개발 트렌드 & 팁**
- 유튜브: 튜토리얼, 고객 사례
- 웨비나: **월간 AI 웹개발 세미나**

### 5.3 파트너십

- 에이전시 리셀러 (20~30% 수익 배분)
- 호스팅 업체 공동 프로모션
- 교육기관 제휴 (AI 웹개발 교육 과정)

## 6. 실행 로드맵

- **Phase 1 (1~6개월)**: MVP 개발(템플릿 10종), 베타 유저 50명
- **Phase 2 (6~12개월)**: 한국 로컬 연동, 유료 고객 100명, 월 매출 2.5억 원
- **Phase 3 (12~24개월)**: 엔터프라이즈 기능 추가, 파트너십 확대, 해외 준비
- **Phase 4 (24~36개월)**: 일본·동남아 진출, IPO 준비, 월 매출 30억 원
